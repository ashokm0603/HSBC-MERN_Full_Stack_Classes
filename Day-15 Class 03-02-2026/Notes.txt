    backend APIs : 
    ---------------
    1. post method (register api )
            http://localhost:4000/api/register

    2. post method (login api )
            http://localhost:4000/api/login
    

    3. post method (add product api )
            http://localhost:4000/api/add-product

    4. get method (get product api )
            http://localhost:4000/api/get-products

    5. delete method (delete product api )
            http://localhost:4000/api/delete-product/:id

    6. put method (update product api )
            http://localhost:4000/api/update-product/:id

    7. get method (get product based on ID api )
            http://localhost:4000/api/get-product/:id
